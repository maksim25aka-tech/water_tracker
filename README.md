💧 Трекер водного баланса – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольного трекера водного баланса на 7 языках программирования.
Каждая версия позволяет вести учёт выпитой воды, устанавливать дневную цель, просматривать статистику и сохранять данные.

📝 Что такое Трекер водного баланса?
Приложение для ежедневного контроля потребления воды. Вы фиксируете каждый приём жидкости (в миллилитрах), программа суммирует объём, показывает прогресс к цели и даёт статистику за день и неделю.

Основные возможности:
Добавление записи – указать объём (мл) и опциональную заметку.

Просмотр записей – за сегодня, за последние 7 дней или все.

Статистика – общий объём за день/неделю, среднее, процент выполнения цели.

Установка дневной цели – по умолчанию 2000 мл.

Удаление записи по ID.

Сохранение / загрузка данных в формате JSON.

✨ Особенности
Интерактивное меню с выбором действий.

Автоматическая временная метка для каждой записи.

Валидация ввода (число > 0).

Цветной вывод (где поддерживается) для наглядности.

Данные сохраняются в файл water_data.json (или эквивалент).

Загрузка данных при старте.

Расширенная статистика: общий объём, среднее за день, прогресс к цели.
